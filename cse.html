Computer Science Engineering Department

About the Department

The Computer Science Engineering (CSE) Department of Ladakh Polytechnic College provides knowledge in computer programming, networking, database management, and software development. The department also has a modern computer laboratory where students can practice coding and technical skills.

Lecturers and Subjects

1. Jigmat Dorjey

Jigmat Dorjey teaches practical and theory subjects related to programming and software development.

  • Operating System Theory (OST)
  • Object Oriented Programming System (OOPS) Lab
  • C Programming Lab
  • Full Stack Development Theory
  • Full Stack Development Lab

2. Tsering Gurmet

Tsering Gurmet teaches important theory subjects in computer science and mathematics.

  • Computer Organization and Architecture (COA)
  • Object Oriented Programming System (OOPS) Theory
  • Discrete Mathematics

3. Padma Angmo

Padma Angmo teaches database and programming theory subjects.

  • Database Management System (DBMS) Theory
  • C Programming Theory

4. Padma Chosphel

Padma Chosphel teaches modern technology and introductory computer subjects.

  • Internet of Things (IoT)
  • Artificial Intelligence and Neural Network (AI & NN)
  • Introduction to Computer and IT

Computer Lab Class

The CSE Department has a computer lab with many computers and internet access. Students attend lab classes every week to practice programming, web development, database work, and networking.

  1. C Programming Practical
  2. OOPS Practical
  3. Full Stack Development Lab
  4. Database and Networking Practice

Student Attendance Notice

Some students did not attend the class regularly. Because of irregular attendance, they may miss important lectures and practical work in the laboratory.

Notice: All students should attend classes regularly and complete their lab work on time.
